How to Get Started

1. Download Extension

Click here to download the SwiftSummarizer zip file.

2. Load Unpacked Extension

Go to chrome://extensions, enable Developer Mode, and click Load Unpacked.

3. Setup Gemini API Key

Get your API key from Google AI Studio and paste it inside the extension settings.

4. Start Summarizing!

Browse any article and run SwiftSummarizer to get your summary instantly!
